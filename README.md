# 实验室简介

北京电子科技学院（BESTI）系统安全与隐私保护实验室（CSP）

# 导师简介

张健毅

- 2012年博士毕业于北京邮电大学计算机科学与技术专业信息安全方向，获工学博士学位。
- 现任北京电子科技学院副教授、路易斯安那大学拉法叶分校博士生导师。
- 曾就职于华为安全部从事机器学习与信息安全结合方面的研究。10年IT领域相关技术研发工作经验。对人工智能、机器学习、信息安全等具有丰富经验。
- 现作为负责人完成/主持国家自然基金委、科技部、国家应急中心、中国电信、农业银行等系列课题10多项，安全方面承担了多项国家安全专项、国家重大科技专项以及党政系统安全顶层设计等工作。带领团队致力于业务支撑、数据挖掘、数据安全方面的教学、科研和成果转化工作。
- 研究成果先后发表在中科院1区、CCF-A 在内的高水平 SCI 与顶级会议。

## 研究方向

- 系统安全：

  无线电安全（GPS伪造、RFIC破解与伪造、智能钥匙伪造、门禁系统伪造、伪基站）；传感器安全（利用EMI、Ultrasonic、激光、磁信号对惯性传感器、麦克风、温湿度传感器等传感器的Spoofing Attack或DoS攻击；检测与防御方法）、二进制安全（Fuzzying与Anti-Fuzzying）

- 人工智能安全

  联邦学习（横纵向联邦的后门攻击、数据投毒攻击；异质模型的数据攻击；成员推断攻击；联邦公平性；抵抗后门攻击的防御方法；联邦原型学习、个性化联邦学习等的安全问题）、大语言模型（越狱攻击、隐语理解力评价、大模型安全评价、生成式内容的检测）

- 密码学与区块链

  安全多方（OT）、同态密码（可GPU加速的同态算法）、可搜索的非对称加密、加密货币溯源与定位、可监管区块链系统

- 应用安全

  验证码对抗与安全、Web应用渗透测试

- 人工智能应用

  大语言模型垂直化应用、数据挖掘、知识增强（RAG）

## 承担项目

- 国自然 《基于内容的图像光影模板学习与美学质量评价关键技术研究》

- 公安部 《面向移动终端隐式页面的恶意内容检测技术研究》

- 公安部 《智能终端 NDEF 协议安全自动化测试机攻击技术研究》

- 教育部 《面向入侵检测的自适应蜜网构建》

- 中央高校 《态势感知关键技术研究》

- 中央高校 《敏感网络定向 APT 攻击的泄密识别与防范策略研究》
